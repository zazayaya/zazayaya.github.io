Hexo

搭建hexo博客

零、简介

hexo是一款基于Node.js的静态博客框架

一、安装git

git下载
PATH环境选择最下面一项(Use Git and opt Unix tools..)

二、安装nojs

nojs下载

三、安装Hexo

Hexo安装文档

# hexo 3.x安装命令
npm install -g hexo-cli

四、创建博客

hexo init <folder>
cd <folder>
npm install

五、修改配置文件

#Site
title: zaza的博客
subtitle: zaza's blog
description: IT技术
author: zaza
language: zh-CN
timezone: Asia/Chongqing

#URL
url: http://zazayaya.github.io

六、使用github发布博客

1、创建新仓库

仓库命名规则： yourgitname.github.io

2、生成key

# 如果生成路径是"//.ssh/github_rsa"，说明不存在HOME变量，需要"右击计算机-属性-高级系统设置-环境变量-xx用户变量"，新建变量HOME，值为：C:\Users\用户名
ssh-keygen -t rsa -C "zyqojj@gmail.com"
# 当前显示类似路径：/c/Users/you/.ssh/id_rsa

3、创建config

# C:\Users\zaza\.ssh\config
host github.com
    user git
    hostname github.com
    port 22
    identityfile C:/Users/zaza/.ssh/id_rsa

4、添加github公钥

# 公钥路径： C:\Users\zaza\.ssh\id_rsa.pub

添加公钥

5、测试github

ssh -Tv github.com

6、安装hexo git模块

# 适用于hexo 3.x
npm install hexo-deployer-git --save

7、修改_config.yml

# Deployment
## Docs: http://hexo.io/docs/deployment.html
deploy:
  type: git
  #repo: https://github.com/zazayaya/zazayaya.github.io.git
  repo: git@github.com:zazayaya/zazayaya.github.io.git
  branch: master

8、发布测试

# hexo new [layout] <title>
# 创建文章
hexo new 新的开始
# 发布文章
hexo d -g

9、网站测试

访问：http://zazayaya.github.io/

10、cname域名

如果你希望将域名更改为自己的域名：

  • 在DNS后台添加对应的域名，并CNAME到http://zazayaya.github.io/
  • 在blog的source目录下创建CNAME文件，并写入被CNAME的域名
  • 修改_config.yml下的URL域名
  • 发表到仓库：hexo d -g
  • 访问新域名看是否成功

11、常见问题

# a、fatal: Not a git repository (or any of the parent directories): .git)
解决方法：删除博客下的.deploy_git目录，重新发布即可
